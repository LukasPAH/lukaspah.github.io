#item_id_aux Calculator for Minecraft Bedrock Edition

This is a calculator to determine the item ids that can be attached to a UI. Minecraft does a special calculation to determine these values and this tool repeats those calculations. This tool will return a value used to go along with #item_id_aux in Minecraft UI files. There are two parts to this tool since the calculation is different: items and blocks. You will need the item or block ID (the DEC number next to the icon on the Wiki) to do this calculation, which can be found at: https://minecraft.gamepedia.com/Bedrock_Edition_data_value.

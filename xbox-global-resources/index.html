About Custom Global Resources on Xbox One

This page is about adding custom global resources to the Xbox One. Why? To use custom textures on Realms, Multiplayer Games, or Servers without requesting the host apply the pack to the world. The following method works due to the way Minecraft handles packs from the Marketplace with the same UUID and version number in manifest.json. Additionally, I added the line

"pack_scope": "global"

to manifest.json to provide additional functionality.


To some, the method going to be presented may seem kind of "hacky". This claim has some validation, as at any time, this method may be patched by either Mojang or Microsoft itself. This method was discovered by me, LukasPlaysGames, in late August 2019. Be careful and read the instructions CAREFULLY.


Download Pack Manifests

Currently available pack manifests are available for download at the following link. If you are unsure which folder to click, choose the Texture Update Beta folder and follow the instructions below.

https://www.mediafire.com/folder/65i1yks3p0719/Manifest_Files#65i1yks3p0719


Written Intructions

***IMPORTANT: These instructions were written for downloading the Texture Update Beta. The process will be the same if you want to use another pack from the store as a base instead. Just download the correct pack from the Marketplace corresponding to the pack manifest you downloaded.***


Step 1

  • Download the Texture Update Beta from the Marketplace.

Step 2

  • Download and place the sample manifest into any resource pack of choice. Manifest downloads are located in the "Download Pack Manifests" section on this page.

Step 3

  • Move the resource pack into the resource_packs folder using the UFO.Transfer method. Detailed instructions can be found at the following link, courtesy of @agentms_:
    • https://www.mcbedrock.com/community/discussion/how-to-install-custom-resource-behavior-packs-on-xbox-ones/.

Step 4

  • Launch Minecraft. Go to global resources. Delete the Texture Update Beta by applying the pack, clicking the green checkmark, then the trash icon. This will delete the beta textures downloaded from the store, but not the custom ones. You should be able to apply your textures using the Texture Update Beta icon in the global resource tab. Therefore, use the texture update icon to activate the custom resources.
